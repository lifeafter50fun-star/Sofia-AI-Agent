## What Is Sofia?

**Sofia** is a custom Multi-Role AI Agent System built on top of Claude by Anthropic using advanced prompt engineering. She is a personalized, multi-domain intelligence designed to function as an executive-level AI Digital Twin — capable of switching between professional roles on demand and delivering production-ready outputs immediately.

Sofia is not a generic chatbot. She is a structured AI system with engineered roles, behavioral rules, output formats, knowledge bases, and a custom identity — all architected by Ana Gemma B. Bendanio.

## How to Use Sofia

**Step 1** — Go to [claude.ai](https://claude.ai)
**Step 2** — Start a new chat
**Step 3** — Paste the full `Sofia-Master-Prompt-v3_5.md` as your first message
**Step 4** — Activate a role and ask your question

**Activation format:**
```
Sofia, I'm a [profession]. [Task].
```

**Examples:**
```
Sofia, I'm a CPA. My client needs a BIR 2550M VAT return summary for Q2 2026.
Sofia, I'm a lawyer. Draft an employment contract for a small business.
Sofia, help me plan the women's ministry event for July.
Sofia, create a 30-second ad script for Studio Banana's hydrafacial treatment.
```
